The Anatomy of a Web Page: 14 Basic Elements

Review the essential elements of a web page, learn what impact each element has and how it contributes to the general user experience. Packed with web design examples.

Header

The header is the upper (top) part of the webpage. Being the area people see before scrolling the page in their first seconds on the website, the header is an element of strategic importance...

  • Basic elements of brand identity (usually a logo)
  • Call-to-action button
  • Links to basic categories of website content
  • Links to the social networks
  • Basic contact information (telephone, email, etc.)
  • Language switcher in multilingual websites
  • Links to interaction with the product (trial version, download, etc.)

It doesnâ€™t mean that all the mentioned elements should be included in one web page header...

CTA Button

A call-to-action (CTA) button is an element of a user interface aimed at encouraging a user to take a certain action...

Effective call-to-action buttons are easy to notice; designers intentionally create them so that website visitors could see them in split seconds...

Hero Section

The hero section is the above-the-fold (pre-scroll) area of the web page containing a strong visual hook...

Footer

The footer is the lower (bottom) part of the web page which usually marks its end. Being another common zone of global website navigation, the footer provides additional links and useful data...

Footers can include:

  • Brand identity signs (company name and logo)
  • Links to support sections (FAQ, Privacy Policy, Terms, etc.)
  • Credits to website creators
  • Contact forms and information
  • Links to social networks
  • Testimonials and badges
  • Certification signs
  • Subscription field or button

As with headers, the footer is not always present, especially with infinite scrolling websites...
